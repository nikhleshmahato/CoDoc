# Codoc
CoDoc means Coding with Documentation. CoDoc combines you Code editor and Notes Editor .
No more need to write your codes and notes in separate file . 

## 🚀 Code + Documentation = CoDoc

**Codeoc** is a platform that combines **coding** and **documentation** together.  
No need to maintain separate files for your code and notes anymore.

Write your:
- 💻 Code
- 📝 Explanations
- 📚 Notes
- ⚡ Ideas
- 🔧 Commands

—all inside a single file.

---
## 🌐 Future Updates

In future updates, online compilers and interpreters will be linked directly with Codeoc.

You will be able to:
- ▶️ Run code directly from your notes
- ⚡ Execute multiple programming languages
- 🧪 Test programs instantly
- 📝 Write explanations beside your code
- 🚫 Avoid switching between different tools and platforms

No need to copy your code into external IDEs or online compilers anymore.

Codeoc aims to make coding, learning, documenting, and testing happen in one unified workspace.
